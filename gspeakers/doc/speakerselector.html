The SpeakerSelector

This is a utility where you can select which speakerlist you want to work with.
 

The speaker selector window
 

Howto use the speakerlist selector

Use this window to select the speakerlist you want to work with.
 

The buttons

Open: Open a speakerlist.
 
Edit speakers...: Press this button to run the SpeakerEditor dialog. Using the SpeakerEditor you can edit various properties of the currently selected speaker list.
