Help needed

Open source projects always need help!!! Theres always something you can do to help. Check the TODO-file for the unstable GSpeakers to see where to start if you want to help. Most issued in the TODO-file is not very difficult. The TODO-file contains mostly improvements to the current source tree.
 
Roadmap
The roadmap contains features that will be added in future releases of GSpeakers.
 
GSpeakers-0.3.*: Bugfix current release.
 
GSpeakers-0.4: Add filter control panel, a simple crossover network design tool.
 
GSpeakers-0.5: Add SPICE-simulations of crossover networks.
 
GSpeakers-0.6: Add part placement design view.
 
GSpeakers-0.7: New features and more improvements.
 
If everything goes as planned I will perhaps reach GSpeakers-0.7 at the end of this year... If you help me we can get there even faster!!!
