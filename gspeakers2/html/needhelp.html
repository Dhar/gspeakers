Help needed

Open source projects always need help!!! There's always something you can do to help. Check the TODO-file for the unstable GSpeakers to see where to start if you want to help. Most issues in the TODO-file is not very difficult. The TODO-file contains mostly improvements to the current source tree.
 
Roadmap (updated 2002-09-14)
The roadmap contains features that will be added in future releases of GSpeakers.
 
GSpeakers-0.6.*: Bugfix current release.
 
GSpeakers-0.7: Improve the SpeakerEditor and move it into a separate tab. Also store the frequency response of a speaker and display the (free air) impedance and (optional) freq response in a plot (or maybe in the same plot). This tab should work as a list of datasheets of speakers.
 
GSpeakers-0.8: A plot of the estimated freq response for the total loudspeaker (driver freq resp. + filter).
 
GSpeakers-0.9: Add part placement design view (only visual view, not interactive).
 
GSpeakers-0.10: GSpeakers should compile and run on windows (this might as well be 0.8 or 0.10...)
 
GSpeakers-0.11: Not really defined yet, probably more advanced features or a first go at the long TODO file.
 
Somewhere down the road: SPICE model of the impedance in an enclosure, airflow in port graph, ... If everything goes as planned I will perhaps reach GSpeakers-0.12 at the end of this year... If you help me we can get there even faster!!!
